### 技巧

 💡断点调试过程中，是运行状态，是以对象的运行类型来执行的。

```
A extend B; 
B b = new A();
b.xx();
```



### 快捷键

+ `F7`：跳入方法内部
+ `F8` ：跳过，住逐行执行代码
+ `shift + F8` ：跳出方法
+ `F9` ：（resume，执行下一个断电）
